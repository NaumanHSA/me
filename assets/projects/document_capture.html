Document Capture SDK is a lightweight, high-performance JavaScript library built for capturing high-quality images of physical documents directly in the browser. The solution ensures documents are precisely detected, cropped, aligned, and classified in real time, enabling smooth digital onboarding and document analysis workflows.

This library is ideal for applications such as Semantic Entity Recognition, Relation Extraction, OCR, and any scenario requiring consistently well-aligned and high-resolution document images.

üåü Project Overview

The library performs live document capture via the user‚Äôs webcam and integrates a two-stage pipeline:

  1. Document Detection and Capture

    • Detects documents in the video feed.
    • Identifies their corners and extracts masks to enable perfect cropping and perspective correction.

  2. Document Type Classification

    • Classifies captured documents into predefined categories (e.g. ID cards, passport, visa).
    • Supports recognition of both known and unknown document types.

All processing is performed client-side, leveraging YOLOv8 for its powerful yet efficient models for object detection, segmentation, classification, and keypoints estimation.
