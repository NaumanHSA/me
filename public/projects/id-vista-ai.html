An advanced, production-ready document analysis engine built with FastAPI and OpenCV. This tool performs end-to-end intelligent document understanding using deep learning models for document detection, type classification, orientation correction, field-level information extraction, and multi-language OCR.

ðŸš€ Overview

This application processes official identification documents such as:

  • Emirates ID (Old/New)
  • GCC ID
  • Passports (50+ countries)
  • Emirates Visas

It performs robust document parsing by:

  1. Detecting and segmenting the document.
  2. Correcting its orientation and aligning it.
  3. Classifying the document type.
  4. Extracting structured information (name, ID, DOB, etc.).
  5. Applying OCR (Optical Character Recognition) for English and Arabic.
  6. Returning processed images and structured metadata in JSON.

ðŸ§  Workflow

The document pipeline consists of the following stages:

Step Description
1. Document Segmentation A YOLO-based model detects the document boundaries and corners.
2. Cropping & Alignment Using corner points, the document is transformed via affine warp to a canonical view.
3. Orientation Correction A classifier checks for rotation and rotates the document upright.
4. Type Classification A document classifier predicts whether it's a passport, visa, ID (new/old, front/back), or unknown.
5. Information Detection Detection models identify key elements (photo, MRZ, signature, text regions).
6. OCR (EN/AR) OCR is applied to the extracted regions using CRNN or PaddleOCR.
7. Response Generation Final structured JSON response is generated, including images and text fields.
