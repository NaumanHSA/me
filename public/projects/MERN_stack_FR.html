The Face Recognition MERN Stack Application is a full-stack web application that enables users to register, recognize, and manage identities through facial images. Designed to demonstrate the power of deep learning integrated with modern web technologies, it seamlessly connects a React front-end with a Node.js backend and MongoDB database.

Face Recognition Pipeline Explained

1. Face Detection

  • Detects faces in images using:
    • SSD MobileNet V1 for fast detection.
  • Outputs bounding boxes around detected faces.

2. Landmark Detection

  • Identifies 68 facial landmarks:
    • Eyes, nose, mouth, jawline.
  • Helps align faces properly despite pose variations.

3. Face Alignment

  • Crops and warps the face image so key landmarks are in standard positions.
  • Ensures consistent embeddings even with different facial angles or expressions.

4. Feature Extraction

  • Uses a deep Convolutional Neural Network similar to ResNet-34.
  • Converts each face into a:
    • 128-dimensional face embedding vector.
  • This vector uniquely represents the person‚Äôs facial features.

5. Face Comparison

  • Compares new face embeddings to stored ones using:
    • Euclidean distance metric.
  • Recognition decision:
    • Small distance ‚Üí same person.
    • Large distance ‚Üí different person.

6. Database Integration

  • Stores:
    • Embeddings
    • User names
    • User emails
  • MongoDB serves as a persistent identity store for the system.
