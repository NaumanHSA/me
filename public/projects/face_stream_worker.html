Face Stream Worker is a robust, production-ready backend system built with FastAPI for managing and processing real-time video streams from IP cameras capable of detecting and tracking faces. Designed with scalability and efficiency in mind, it provides asynchronous operations, multiprocessing for handling multiple video streams concurrently, and optional Redis integration for event broadcasting.

ðŸŒŸ Project Overview

The system allows users to dynamically register and manage IP cameras and process live video streams for face detection and tracking. It offers a set of HTTP API endpoints for registering cameras, retrieving information, and handling deregistration. The platform is ideal for real-time surveillance, analytics, or any application requiring automated video stream processing.

At its core lies a highly extensible processing engine capable of handling large-scale streaming operations while maintaining performance and resilience against connection failures.
